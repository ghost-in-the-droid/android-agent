# Ghost Showcase — Brand Assets

Source of truth: `scripts/showcase/generate_brand_assets.py` (all PNGs are
code-generated — regenerate instead of hand-editing; there is no separate
Figma file). Palette mirrors `site/src/styles/custom.css`.

```bash
python3 scripts/showcase/generate_brand_assets.py            # v1.3.0 defaults
python3 scripts/showcase/generate_brand_assets.py --version 1.4.0
```

## Compositing a screen recording into a device frame

`frames.json` gives, per device, the `screen_rect` `[x, y, w, h]` inside the
1920x1080 canvas and the source recording resolution. Scale the recording to
`w x h`, position it at `(x, y)`, then overlay the frame PNG on top (the frame
covers the corner rounding and camera cutout):

```bash
X=$(jq .pixel8.screen_rect[0] frames.json); Y=$(jq .pixel8.screen_rect[1] frames.json)
SW=$(jq .pixel8.screen_rect[2] frames.json); SH=$(jq .pixel8.screen_rect[3] frames.json)
ffmpeg -i recording.mp4 -i frame-pixel8.png -filter_complex \
  "color=c=0x070b08:s=1920x1080[bg];[0:v]scale=${SW}:${SH}[scr];\
   [bg][scr]overlay=${X}:${Y}[comp];[comp][1:v]overlay=0:0" \
  -c:v libvpx-vp9 -b:v 0 -crf 36 framed.webm
```
