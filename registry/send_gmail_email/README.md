# Send Gmail Email

> Auto-generated skill for sending emails via the Gmail Android app.

## Install

```bash
android-agent skill install send_gmail_email
```

## Actions

No standalone actions -- this skill uses a recorded macro workflow.

## Workflows

| Workflow | Description | Steps |
|----------|-------------|-------|
| `recorded` | Send an email with subject and body | open_compose, fill_subject, fill_body, tap_send |

## Tested On

|--------|---------|--------|
| Various devices | 12+ | Pass |
