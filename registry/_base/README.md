# Base Actions

> Shared actions usable by any Android app skill. These are the building blocks that all other skills depend on.

## Install

Included by default with Android Agent -- no separate installation needed.

## Actions

| Action | Description |
|--------|-------------|
| `tap_element` | Tap a UI element by name or locator |
| `swipe_direction` | Swipe in a direction (up/down/left/right) |
| `type_text` | Type text into the focused input field |
| `wait_for_element` | Wait for a UI element to appear on screen |
| `launch_app` | Launch an Android app by package name |
| `take_screenshot` | Capture a screenshot and save to file |
| `dismiss_popup` | Detect and dismiss common popup dialogs |
| `press_back` | Press the Android back button |
| `press_home` | Press the Android home button |

## Workflows

No workflows -- this skill provides primitive actions only.
