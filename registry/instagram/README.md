# Instagram

> Instagram automation skill -- skeleton with core actions.

## Install

```bash
android-agent skill install instagram
```

## Actions

| Action | Description |
|--------|-------------|
| `open_app` | Launch Instagram and wait for home screen |
| `navigate_to_profile` | Navigate to the Profile tab |
| `tap_search` | Tap the search/explore icon |
